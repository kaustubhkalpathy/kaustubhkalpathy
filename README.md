## 🚀 Projects

### 🏦 [Loan Underwriting Copilot](https://github.com/kaustubhkalpathy/loan-underwriting-copilot)
Credit risk prediction system for loan officers — classical ML meets serverless cloud deployment.
- **LightGBM** model trained on 307,511 real loan applications → **ROC-AUC 0.759**
- Per-prediction **SHAP explainability** with plain-English toggle
- Deployed on **AWS Lambda** (containerised, ~8ms warm inference)
- Streamlit UI with What-If Analyzer, Batch Assessment, PDF reports
- **Tech:** Python · LightGBM · SHAP · AWS Lambda · S3 · ECR · Bedrock · Docker · Streamlit

### 📈 [Market Intelligence Agent](https://github.com/kaustubhkalpathy/market-intelligence-agent)
AI-powered financial analysis copilot that autonomously fetches live market data and answers investment questions.
- **ReAct agentic pattern** — LLM autonomously decides which of 7 tools to call
- Powered by **Amazon Bedrock** (Nova Lite / Nova Pro / Claude) via Converse API
- Real-time stock data, news, sector analysis, portfolio tracking
- Full observability dashboard + 38 automated tests
- **Tech:** Python · Amazon Bedrock · yfinance · Streamlit · Plotly · Docker · ECS Fargate
